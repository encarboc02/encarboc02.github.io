01

Análisis de reacciones adversas a Ozempic

Análisis de datos FAERS de la FDA para identificar patrones de reacciones adversas a Ozempic, explorando características demográficas (edad, sexo), número de medicamentos concomitantes y resultados graves. Implementé modelos predictivos (Random Forest y regresión logística) y visualicé las principales reacciones adversas, aportando información sobre factores de riesgo. Proyecto orientado a farmacovigilancia y análisis de datos clínicos.

PythonMLRandom ForestFAERSFarmacovigilancia

02

Clinical Trials BI — Ensayos clínicos con Python y Power BI

Análisis de datos de ensayos clínicos intervencionales de ClinicalTrials.gov, extrayendo y procesando información mediante Python para identificar patrones en la investigación clínica global: países, patrocinadores y tamaño de los estudios. Desarrollé un dashboard interactivo en Power BI para visualizar tendencias y facilitar la exploración de datos. Proyecto enfocado en Business Intelligence aplicado al sector sanitario y farmacéutico.

Power BIPythonData VisualizationIndustria Pharma

03

Pipeline para análisis de expresión génica diferencial

Pipeline reproducible en RMarkdown para análisis de RNA-Seq del hipocampo de ratón, incluyendo preprocesamiento y control de calidad (FastQC, Salmon), análisis de expresión diferencial (DESeq2), exploración de datos (PCA, heatmaps) y anotación y enriquecimiento génico (biomaRt). Proyecto reproducible orientado a entrenamiento en bioinformática y análisis transcriptómico.

RRMarkdownDESeq2RNA-SeqbiomaRtFastQC

04

Aplicación web para cribado de fármacos — TFG

Proyecto de Fin de Grado en Biotecnología. Herramienta web para asistir a investigadores en las etapas iniciales del descubrimiento de fármacos: consulta de dianas terapéuticas en ChEMBL y análisis de descriptores fisicoquímicos (regla de Lipinski). Diseñada para reducir el tiempo y los costes en las fases iniciales de I+D farmacéutica.
